# 生产外部凭据交接清单

本清单只记录真实联调需要的外部输入，不要求把密钥写进仓库。

## 管理后台

- `ADMIN_INITIAL_USERNAME`：首次管理员账号。
- `ADMIN_INITIAL_PASSWORD`：首次管理员密码，放 Edgespark secret。
- `ADMIN_PASSWORD_PEPPER`：后台密码加盐，放 Edgespark secret。

## DeepSeek 写信

- `LETTER_PROVIDER=deepseek`
- `DEEPSEEK_MODEL=deepseek-v4-pro`
- `DEEPSEEK_BASE_URL=https://api.deepseek.com`
- `DEEPSEEK_API_KEY`：放 Edgespark secret。

生产验收：

```powershell
$env:XIABI_VERIFY_DEEPSEEK="1"
npm run verify:production
```

## 微信支付

- `PAYMENT_PROVIDER=wechat`
- `PUBLIC_BASE_URL=https://immortal-sponge-1728.edgespark.app`
- `PAYMENT_NOTIFY_URL=https://immortal-sponge-1728.edgespark.app/api/webhooks/wechat-pay`
- `WECHAT_PAY_APP_ID`：公众号或支付场景对应 AppID。
- `WECHAT_PAY_MCH_ID`：微信支付商户号。
- `WECHAT_PAY_PRIVATE_KEY`：商户 API 私钥，放 Edgespark secret。
- `WECHAT_PAY_CERT_SERIAL_NO`：商户 API 证书序列号，放 Edgespark secret。
- `WECHAT_PAY_API_V3_KEY`：API v3 密钥，放 Edgespark secret。
- `WECHAT_PAY_PLATFORM_PUBLIC_KEY`：微信支付平台公钥，放 Edgespark secret。
- `WECHAT_MP_APP_SECRET`：公众号网页授权 secret，微信内 JSAPI 支付需要，放 Edgespark secret。

生产验收分两步：

```powershell
$env:XIABI_VERIFY_PAYMENT_CREATE="1"
npm run verify:production
```

然后在微信内完成一笔真实小额付款，确认订单变为已支付、权益流水生成、重复回调不重复发权益。

## 阿里云短信

- `SMS_PROVIDER=aliyun`
- `SMS_ALIYUN_SIGN_NAME`：短信签名。
- `SMS_ALIYUN_TEMPLATE_CODE`：短信模板 Code。
- `SMS_API_KEY`：阿里云 AccessKey，放 Edgespark secret。
- `SMS_API_SECRET`：阿里云 AccessSecret，放 Edgespark secret。

生产验收：

```powershell
$env:XIABI_VERIFY_SMS_PHONE="可接收验证码的手机号"
npm run verify:production

$env:XIABI_VERIFY_SMS_CODE="收到的6位验证码"
npm run verify:production
```

## MiniMax 语音

- `VOICE_PROVIDER=minimax`
- `VOICE_API_KEY`：MiniMax API Key，放 Edgespark secret。
- `MINIMAX_VOICE_ID`：智多星说话音色，例如 `moss_audio_7a15101b-4f96-11f1-b3fb-821d30892425`。

生产验收：

```powershell
$env:XIABI_VERIFY_TTS="1"
npm run verify:production
```

## 语音输入 ASR

当前用户端优先使用浏览器语音识别；浏览器不支持时，服务端会把录音转发到可配置 ASR 接入位。

- `VOICE_ASR_PROVIDER`：供应商标识。
- `VOICE_ASR_ENDPOINT`：服务端 ASR HTTP 接口。
- `VOICE_ASR_MODEL`：可选模型名。
- `VOICE_ASR_API_KEY`：ASR Key，放 Edgespark secret；如果 ASR 和 TTS 共用 Key，也可以复用 `VOICE_API_KEY`。

生产验收：

```powershell
$env:XIABI_VERIFY_ASR_AUDIO="D:\path\to\sample.wav"
npm run verify:production
```

## 验收结论口径

- 没有真实 verifier 环境变量时，只能证明基础线上运行正常。
- 完整真实运行状态必须完成：后台登录、DeepSeek 生成、短信绑定、微信付款回调发权益、MiniMax TTS、ASR 音频样本。
